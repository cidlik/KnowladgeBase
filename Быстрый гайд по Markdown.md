### Заметки

| Syntax | Description |
| ----------- | ----------- |
| Heading | # H1 <br/> ## H2 <br/> ### H3 |
| Bold | ** bold text ** |
|Italic|*italic text*|
| Ordered list | 1. First item <br/> 2. Second item <br/> 3. Third item |
| Blockquote | > some phrase |
| Code | `one backtick` <br/> ``` three backticks ``` |


### Списки

##### Неотсортированный список
- Элемент а
- Элемент б
- Элемент в

##### Отсортированный список
1. Первый элемент
2. Второй элемент
3. Третий элемент

##### Вложенные списки
1. Первый элемент
2. Второй элемент
	1. Первый подэлемент
	2. Второй подэлемент
3. Третий элемент
	- Очередной подэлемент
